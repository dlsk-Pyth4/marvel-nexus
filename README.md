# 🦸‍♂️ Marvel Nexus - Application Fullstack

Marvel Nexus est une plateforme interactive pour les fans de l'univers Marvel, permettant de consulter des fiches personnages et de partager des théories via un espace communautaire sécurisé.

## 🚀 Fonctionnalités
- **Interface Dynamique** : Développée avec Next.js et Tailwind CSS.
- **Serveur API** : Backend robuste sous Express.js.
- **Base de données** : Persistence des messages via SQLite.
- **Sécurité** : Protection contre les injections SQL (requêtes préparées) et failles XSS (sanitization).

## 🛠️ Installation et Lancement

1. **Cloner le projet**
2. **Installer les dépendances** : `npm install`
3. **Lancer le Frontend** (Port 3000) : `npm run dev`
4. **Lancer le Backend** (Port 3001) : `node server.js`
